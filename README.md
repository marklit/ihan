# IHAN Client

[IHAN](https://www.ihan.ee/) Client for feeding and back filling log files.

## Requirements

Python 2.7, Python 3.3, 3.4 or 3.5 are supported.

## Installation

```python
$ virtualenv ~/.ihan
$ source ~/.ihan/bin/activate
$ pip install --upgrade ihan
```
